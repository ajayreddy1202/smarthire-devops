# SmartHire — Job Application Tracker

A full-stack job tracking application built with Python Django, Docker, and AWS.

## 🌐 Live Demo
http://13.126.107.226:8000

## 🛠️ Tech Stack
- **Backend:** Python, Django, Django REST Framework
- **Database:** MySQL
- **Containerization:** Docker
- **Cloud:** AWS EC2 (Ubuntu)
- **CI/CD:** GitHub Actions

## ✅ Features
- User Authentication (Register/Login)
- Add, Update, Delete Job Applications
- Track application status
- Clean REST API
- Responsive UI

## 🚀 Deployment
- Containerized with Docker
- Deployed on AWS EC2 Ubuntu server
- Automated CI/CD via GitHub Actions
- Publicly accessible 24/7

## 📦 Run Locally
```bash
git clone https://github.com/ajayreddy1202/smarthire-devops.git
cd smarthire-devops
docker build -t smarthire .
docker run -p 8000:8000 smarthire
```
